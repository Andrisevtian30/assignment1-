# Laporan Assignment 1

## Deskripsi Program
Program ini digunakan untuk menghitung umur seseorang berdasarkan tanggal lahir yang diinputkan oleh user.

## Fitur Program
- Menghitung umur dalam tahun
- Menghitung umur dalam bulan
- Menentukan hari dalam minggu

## Contoh Input
Tahun: 2005  
Bulan: 5  
Tanggal: 10  

## Contoh Output
Umur (tahun): 19  
Umur (bulan): 230  
Hari lahir: Selasa  

## Penjelasan
Program menggunakan library `<ctime>` untuk mendapatkan waktu saat ini dan menghitung selisih umur serta menentukan hari dalam minggu.
